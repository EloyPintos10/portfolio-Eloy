# Portfolio personal realizado con React.js

Maquetado completo responsive de portfolio personal, realizado con React.js totalmente.

## Librerias Utilizadas

1- React js

2- React Bootstrap

3- React Icons

4- Git

5- Git Hub




### Para poder obtener el repositorio debes hacer lo siguiente:

1- Clonar el repositorio en tu PC - git clone

2- Realizar la instalacion de las dependencias - npm install

3- Lanzar el proyecto en la terminal -  npm start
